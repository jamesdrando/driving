# driving

Browser-first 3D driving toy built with `Three.js`, `Rapier`, `TypeScript`, and `Vite`.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run test`

## Prototype scope

- Third-person arcade buggy
- Deterministic generated desert valley
- Keyboard and touch controls
- Minimal HUD with pause and reset

## Controls

### Desktop

- `W` / `Up`: throttle
- `S` / `Down`: reverse
- `A` / `Left`: steer left
- `D` / `Right`: steer right
- `C`: recenter camera
- `R`: self-right / reset
- `Esc` or `P`: pause

### Touch

- Left and right steering pads
- Go and reverse pedals
- Pause button in the HUD
